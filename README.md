
# techtalks-realtimeprice-availabilitymap

## Description
This project is a map-based platform designed for Lebanon and the MENA region to help users find real-time prices and availability of essential items such as groceries, fuel, medicine, and basic electronics. Due to frequent price changes and unstable supply, people often waste time and resources searching between stores. The platform aims to support smarter purchasing decisions before leaving home.

## Key Features
-Real-time price updates submitted by users or shops.
-Shops can claim their listings and manage products.
-Filter products by category, shop, or price range.
-Search for shops or products based on location or distance.
-Highlight products with price or availability changes.
-Map integration for shop locations.
-Notifications for price drops or low stock.
-User profiles for tracking favorite shops or products.
-Dynamic data updates

## Technology Stack
- Frontend: Next.js
- Backend: Java

### versions 
- Java 17
- Spring Boot 4.0.2
- Maven (via Maven Wrapper)
- H2 Database (in-memory)
- Runs on port 8080

## How to Run the Backend

### Requirements
- Java 17

> Maven is NOT required (Maven Wrapper is included)

### Run backend
on terminal:
- cd backend
- ./mvnw spring-boot:run
- Backend runs on:
  http://localhost:8080
