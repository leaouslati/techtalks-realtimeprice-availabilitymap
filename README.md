
# techtalks-realtimeprice-availabilitymap

## Description
This project is a map-based platform designed for Lebanon and the MENA region to help users find real-time prices and availability of essential items such as groceries, fuel, medicine, and basic electronics. Due to frequent price changes and unstable supply, people often waste time and resources searching between stores. The platform aims to support smarter purchasing decisions before leaving home.

## Key Features
Real-time price updates submitted by users and shops
Shop management dashboard for product control
Search by product name, shop, or category
Filter by price range and availability
Distance-based shop search
Interactive map integration
Highlighted price or stock changes
Notification system for price drops or updates
Dynamic data updates (real-time support ready)

## System Architecture
Frontend: Next.js
Backend: Java (Spring Boot)
Database: H2 (development)
